How to start




ADMIN

    Home Page - Displays a dashboard of Users and Usage reports.
    Company Profile - Add and edit company profile details.
    Change Password - Change password periodically.
    Sub User - Additional login to your domain that you can create and grant limited access.




RECRUITMENT MANAGEMENT SYSTEM

    Home Page - Displays a dashboard of CV classification count, as per the Auto Search Assistant preferences. This can be modified by clicking on the speaker button or Edit Auto Assistant link.
    Create Vacancy - Enter Vacancy details and internal HR contact and project details and hit Post button. This will create a vacancy in the RMS.
    Manage Vacancy - Track recommended and suitable CVs or make any amends to requirements accordingly. Results refresh instantaneously.
    Candidate Status - Check candidate profiles who have applied for the vacancy, and update status accordingly.




JOB PORTAL

    Home Page - Displays a dashboard of Users and Usage reports.
    Job by Location - Search the job based on Location
    Job by Industry - Search the job based on Industry
    Advanced Search - Search for the job and View the details of candidates.




HUMAN RESOURCE

    Home Page - Displays a dashboard of Users and Usage reports.
    Employee Details - Use this function to add, edit and track all employee records.
    Attendance Settings - Track daily and monthly employee attendance. Also track late hours of employees.
    Holiday List - Add, edit and track Holidays listed for the year.




PAYROLL

    Home Page - Displays a dashboard of Users and Usage reports.
    Loan - Add, edit and track details of Loan availed by employees.
    Settings - Add, edit and track compensation, payroll and compensation deduction settings
    Salary Computation - Extract and track salary statements of employees.




CUSTOMER RELATIONSHIP MANAGEMENT

    Home Page - Displays a dashboard of Users and Usage reports.
    Client list - Add, edit and track Clients.
    Contact Details - Add, edit and track Contacts.
    Leads - Add, edit and track Leads.
    Prospects - Add, edit and track Prospects.
    Forecast - Tabulated Report of Sales Forecast.
    Services - Edit and track Servicess.
